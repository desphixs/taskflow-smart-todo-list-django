# Build TaskFlow: A Smart To Do List

### Project Description

You are going to build TaskFlow a very simple but smart to do list where users can track their daily goals. First you will set up a database table that includes a true or false switch to track if a task is done. Next you will use Django built in forms to easily and safely accept new tasks from the user. After that you will write Python logic to cross off items when they are clicked updating their status in the database instantly. Finally you will add a quick filter so users can hide their finished work and focus only on what is left. You will build all of this using function based views so you understand exactly how data moves and changes.
